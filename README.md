•	Carry
o	The return generated form holding a financial instrument if markets don´t move
	E.g. For a stock the margin you pay to lend the money vs. the dividend it pays, so if spot stays unchanged, you still have a pos/neg return
	For options the theta effect from an investor lens
	From a Desk lense, it´s the theta effect (inverse position), plus the funding cost of maintaining the hedge
	For FX, you borrow one currency and invest in the other, can be positive carry (of course without account any potential moves)
•	Bond repack

•	Inverse Floater
o	As opposed to paying Reference Rate + Spread, this pays a fixed Rate minus a leverage Multiplier * the Reference Rate
o	So when the Euribor falls, the Coupon rises and when it rises the Coupon falls, opposite to what would happen with a Floater
o	So you can use this if you want a fixed rate and are betting on falling Rates
